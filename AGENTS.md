# AGENTS.md

## Product

We are building Closure, a life-admin radar app.

Closure helps users reduce hidden cognitive load from unresolved obligations such as deadlines, forms, appointments, bills, applications, renewals, and promises.

The app turns messy user input into clear obligation cards.

## MVP

The first MVP should have:

1. A capture page where users paste messy text.
2. A review page showing extracted obligation cards.
3. A dashboard showing open obligations.
4. Fake data first.
5. No database at the beginning.
6. No OpenAI API call at the beginning.
7. No Gmail integration at the beginning.

## Product principles

- Reduce mental load.
- Do not create another noisy productivity app.
- Fewer, better notifications.
- Never invent deadlines.
- Always show uncertainty.
- The user should review AI-extracted obligations before they become active.
- Privacy is a core feature.
